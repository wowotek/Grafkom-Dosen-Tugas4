## Cara _Compile_
- GNU/Linux Distro Ubuntu 18.04
    - Prasyarat Kompilasi
        1. _**Makefile**_
        2. _GNU C/C++ Compiler Toolkit (**GCC**)_
    - Kompilasi
        ``` bash
        # bersihkan file jika file sudah ada
        make clean
        # compile project
        make all
        # eksekusi executable tugas4
        ./build/bin/tugas4.exec
        ```
